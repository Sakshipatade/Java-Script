### JavaScript Fundamentals & Practice

This repository contains my JavaScript learning journey with topic-wise code examples, covering core concepts from basics to advanced.

### Topics Covered

* Data types and comparisons
* Strings, numbers, and date/time
* Arrays and array methods (slice, splice, map, filter, reduce)
* Objects and functions (including arrow functions, IIFE, closures)
* Scope (global & local) and `this` keyword
* Control flow (switch, logical operators)
* Loops (for, for...of, for...in, forEach)
* Truthy & falsy values
* Pattern and table printing

### Purpose

This repository is created for practicing and strengthening JavaScript fundamentals, with clear examples for each concept.

### Usage

You can explore each file to understand individual concepts and run them using Node.js or in a browser console.
